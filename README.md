> Niko is a Discord Bot with a lot of features to help you manage your server

## Requirements

- **Node >= 10.15.0**
- **Typescript**
- **Redis**
- **NPM** or **Yarn**
- A good code editor like [**Visual Studio Code**](https://code.visualstudio.com/download) (Just for dev)

## Usage

```sh
npm run start
```
